================================================================
  اتصال به Claude Code Server  —  راهنمای سریع
================================================================

⚠️ کل این پوشه را نگه دار (ساختار client/ و server/ را به‌هم نزن).
   فقط connect.bat به‌تنهایی کار نمی‌کند — به فایل‌های کنارش نیاز دارد.

----------------------------------------------------------------
پیش‌نیازها (فقط یک‌بار روی لپ‌تاپ هر نفر)
----------------------------------------------------------------

ویندوز:
  1) VSCode را نصب کن: https://code.visualstudio.com/
  2) در VSCode این دو اکستنشن را نصب کن:  Remote - SSH  +  Claude Code
  3) OpenSSH Server: connect.bat خودش نصبش می‌کند — فقط بار اول روی فایل
     راست‌کلیک کن و "Run as administrator" بزن، و پنجرهٔ UAC را Yes کن.

مک / لینوکس:
  1) VSCode + اکستنشن‌های  Remote - SSH  و  Claude Code
  2) Remote Login (SSH server): connect.sh خودش روشنش می‌کند — فقط بار اول
     پسورد مک را می‌پرسد. (اگر نشد: System Settings -> General ->
     Sharing -> Remote Login = ON ؛ لینوکس: sudo systemctl enable --now ssh)

----------------------------------------------------------------
اولین ورود — تغییر اجباری پسورد (مهم)
----------------------------------------------------------------
بار اول، یک‌بار با SSH ساده وارد شو تا سیستم مجبورت کند رمز را عوض کنی،
بعد connect را اجرا کن:
    ssh username@192.168.210.240
قانون رمز: حداقل ۱۰ کاراکتر، از ۳ نوع (حرف بزرگ/کوچک/عدد/نماد)،
بدون تکرارِ بیش از ۳ تایی.  مثال درست:  MyWork@2025

----------------------------------------------------------------
اتصال
----------------------------------------------------------------
ویندوز:  بار اول روی  client\windows\connect.bat  راست‌کلیک -> Run as administrator
         (دفعه‌های بعد دابل‌کلیک ساده کافی است)
مک/لین :  cd client/mac  و  bash connect.sh

  • بار اول: یوزرنیم سرورت و مسیر پوشهٔ لوکال (مثلاً  D:\Smart ) را می‌پرسد،
    و یک‌بار رمز سرور را برای نصب کلید می‌خواهد.
  • هر بار بعد: فقط اجرا کن و شمارهٔ پروژه را بزن — با کلید وصل می‌شود.

داخل ترمینال VSCode فقط بزن:   claude     (نیازی به لاگین نیست)
  • گفتگوی قبلیِ همان پوشه خودکار ادامه پیدا می‌کند.
  • برای شروع گفتگوی تازه:   CLAUDE_FRESH=1 claude

----------------------------------------------------------------
اگر یوزرنیم یا رمزت عوض شد
----------------------------------------------------------------
  • فقط رمز عوض شد  → هیچ کاری لازم نیست؛ connect با کلید وصل می‌شود.
       (اگر ادمین کلیدت را هم پاک کرده باشد، connect یک‌بار رمز جدید را
        می‌پرسد و کلید را از نو نصب می‌کند — خودش درست می‌شود.)
  • یوزرنیم عوض شد  → دوباره پیکربندی کن:
       ویندوز:  connect.bat -Setup       مک/لین:  bash connect.sh --setup

----------------------------------------------------------------
نکته‌ها
----------------------------------------------------------------
  • باید از قبل روی سرور برایت اکانت ساخته شده باشد (با ادمین هماهنگ کن).
  • اگر VSCode پوشه را خالی نشان داد، چند ثانیه صبر کن یا یک ترمینال در VSCode
    باز کن (خودکار mount می‌شود). اگر مسیر اشتباه باشد پیام واضح می‌بینی.
  • وقتی VSCode را ببندی، تونل خودکار قطع می‌شود — دفعهٔ بعد فقط connect را بزن.
  • به مشکل خوردی؟ به ادمین بگو.
