Claude Code Server - Client Package
=====================================

WINDOWS
-------
1. Copy this folder (claude-code-client-...) somewhere convenient (e.g. Desktop).
2. Open the windows\ folder and double-click connect.bat.
3. First run: asks for your server username and project path, then prompts for
   your server password once to install the SSH key.
4. Every run after: just double-click - VSCode opens on your project automatically.

MAC
---
1. Copy this folder somewhere convenient.
2. In Terminal:
       bash mac/connect.sh
3. First run: asks for server username, project path, and server password once.

Requirements
------------
- Visual Studio Code
- Remote - SSH extension
- Claude Code extension
- VPN connection to the company network (for remote work)

Questions? Contact the admin (smart).
